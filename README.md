# UNBND - Bring Artist x Brand Summit

HTML / CSS / JS website

## Getting Started

Clone or download the files
`https://github.com/dripping-agency/unbnd-bring.git`

### Initialise 
`npm init`

### Compile CSS
`npm run build`

### Compile JavaScript
`npx webpack`
 
### Run local server
`lite-server --baseDir="dist"`
