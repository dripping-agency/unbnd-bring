# UNBND - Bring Artist x Brand Summit

HTML / CSS / JS website

## Getting Started

Clone or download the files
`https://github.com/dripping-agency/unbnd-bring.git`

### Initialise 
`npm init`

### Compile CSS
`npm run build`

### Compile JavaScript
`npx webpack`
 
### Run local server
`lite-server --baseDir="dist"`


## Features

### Speakers

Within the `/dist/speakers` folder, all the content has a html file for each speaker. 

```html
<div class="row">
  <div class="col">
    <figure>
      IMAGE
    </figure>
  </div>
  <div class="col">
    <h2>NAME</h2>
    <h3>TITLE</h3>
    <p>
     DESCRIPTION
    </p>
  </div>
</div>
```
